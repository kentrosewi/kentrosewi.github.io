Sk8 Street - Made with Unity

This game started off as a weekend project but I soon realized that by fleshing out a few of the mechanics I would be able to have a game that was fun to pick up and play every once in a while.

You have the ability to customize your board's deck, trucks, and wheels to create colorful combinations.

The levels are randomly generated so you won't be seeing the same layout twice which adds a bit of replay-ability.

Proceeding through the levels will unlock new moves that your character performs automatically when jumping obstacles. These include the kick-flip, heel-flip, pop shove-it, and reverse pop shove-it. More tricks such as full character rotation or flips could be added and combined with the already implemented tricks.

Feature Request Project

A sample project which allows for feature requests to be submitted by clients.


This application was done outside of work and allows users to submit feature requests for different clients as well as different product areas. It could easily be extended to allow deletion of feature requests, addition of new clients, product areas, and more.

Technology used included the following:

  • JavaScript with Knockout.js, jQuery
  • Bootstrap CSS
  • Python 2.7 and Flask for back-end API
  • SQLAlchemy and MySQL relational database management system
  • Developed in Docker with an Ubuntu instance
  • Amazon Web Services for hosting

This was a great learning opportunity as I had never worked with most of these frameworks/technologies before. My main goal of the project was to learn about new ways of creating applications, and at the same time create clean and reusable code/components.
